# Faux compte bancaire interactif 💳

## Description
Faux compte bancaire interactif pour tester HTML, CSS et JavaScript avec solde, dépôt, retrait et transferts entre comptes. Projet éducatif pour apprendre à créer une application web simple et dynamique.

## Instructions pour tester
1. Ouvrir le fichier `index.html` depuis le navigateur sur smartphone ou PC.
2. Sur l’écran PIN, entrez `0000` pour continuer.
3. Créez un compte avec un nom et un mot de passe.
4. Connectez-vous avec vos identifiants.
5. Testez le dépôt, retrait et transferts entre comptes.
6. L’historique des opérations est affiché automatiquement.

## Remarque
⚠️ Ce projet est **strictement éducatif**.  
Tout est stocké localement dans le navigateur.
